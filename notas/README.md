# Notas destiladas

Notas limpias y estructuradas por sección. Escritas después de leer + ejercicios, pensadas para:
- Yo mismo en 6 meses cuando repase.
- Alguien externo que esté estudiando lo mismo.
- Insumo para los posts semanales.

## Estructura

```
notas/
├── axler/           notas destiladas por sección de Axler (01A-Rn-Cn.md, ...)
├── blitzstein/      notas destiladas por capítulo de Blitzstein
└── (otras fuentes según avance el plan)
```

## Formato estándar

Cada archivo sigue la plantilla:
1. **Definiciones clave** (en mis palabras).
2. **Conceptos que entendí bien.**
3. **Dudas resueltas** (con la respuesta).
4. **Dudas que quedan abiertas.**
5. **Conexiones mentales** (analogías, puentes a cosas que ya sé).
6. **Ejercicios hechos.**

## Filosofía

- Escribirlas es más importante que leerlas.
- Si la versión destilada no se entiende a sí misma, el tema no está aprendido todavía.
- No son resumen del libro — son **mi entendimiento** del libro.
